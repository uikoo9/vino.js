# vico.js

Enterprise-grade Node.js backend framework.

## Features (Planned)

- Modular architecture with plugin system
- Built-in dependency injection
- TypeScript-first
- Decorator-based routing and middleware
- Configuration management
- Logging and error handling
- Database integration support
- Authentication and authorization
- Monorepo structure powered by Lerna + npm workspaces

## Getting Started

```bash
npm install
npm run build
```

## Scripts

| Command            | Description                       |
| ------------------ | --------------------------------- |
| `npm run build`    | Build all packages                |
| `npm run lint`     | Build + Prettier + ESLint         |
| `npm run prettier` | Check formatting                  |
| `npm run eslint`   | Lint code                         |
| `npm run cz`       | Commitizen (conventional commits) |
| `npm run update`   | Update dependencies via ncu       |
